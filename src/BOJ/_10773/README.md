# BOJ 10773 - 제로

## 🔍 문제 요약
정수 K가 주어지고, 이후 K개의 정수가 순서대로 입력된다.

- 입력된 값이 0이 아닐 경우 → 해당 값을 기록한다.
- 입력된 값이 0일 경우 → 가장 최근에 기록한 값을 삭제한다.

모든 입력이 끝난 후 기록되어 있는 값들의 합을 출력하는 문제이다.

---

## 🧠 풀이 아이디어
가장 최근에 입력된 값을 삭제해야 하므로 LIFO 구조가 필요하다.  
따라서 Stack 자료구조를 사용하면 문제를 쉽게 해결할 수 있다.

- 숫자가 0이 아니면 스택에 push
- 숫자가 0이면 가장 최근 값을 pop
- 모든 입력이 끝난 후 스택에 남은 값들을 모두 더해 결과를 출력한다.

---

## 🔎 핵심 로직
1. 스택을 생성한다.
2. K개의 입력을 순회하면서
    - 입력값이 0이면 스택에서 pop
    - 입력값이 0이 아니면 스택에 push
3. 입력이 끝난 후
    - 스택에 남아있는 값들을 모두 더해 출력한다.

---

## ⏱️ 복잡도 분석
- 시간 복잡도: O(K)

- 공간 복잡도: O(K)

---

## ⚠️ 개선 포인트
- 스택을 연습하기 위해 스택으로 풀이했지만, 이 문제는 스택을 사용하지 않고도 해결할 수 있다.
- 입력을 처리하면서 합을 바로 계산하면 마지막에 스택을 순회하는 반복문을 줄일 수 있어 더 간결한 코드로 구현할 수 있다.
