# BOJ 12605 - 단어순서 뒤집기

## 📌 문제 정보
- 플랫폼: BOJ
- 문제 번호: 12605
- 문제 이름: 단어순서 뒤집기
- 언어: Java

## 🔍 문제 요약
한 줄에 주어진 문장의 **단어 순서를 거꾸로 출력**하는 문제.  
각 테스트 케이스마다 결과를 출력해야 한다.

## 🧠 풀이 아이디어
- 문장을 공백 기준으로 분리
- 단어들을 **뒤에서부터 앞으로** 출력
- 테스트 케이스 번호를 함께 출력

## 🔎 핵심 로직
1. 테스트 케이스 개수 T 입력
2. 각 줄을 문자열로 입력받음
3. 공백 기준으로 split
4. 배열을 뒤에서부터 순회하며 출력

## ⏱️ 복잡도 분석
- 시간 복잡도: O(N)
- 공간 복잡도: O(N)

## ⚠️ 주의할 점
- `nextInt()` 후 `nextLine()` 처리 주의
- 출력 형식: `Case #x:` 반드시 맞출 것

- 정상 통과
