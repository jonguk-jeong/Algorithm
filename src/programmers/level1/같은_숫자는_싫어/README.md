# 같은 숫자는 싫어

## 🔍 문제 요약
주어진 배열에서 **연속으로 등장하는 같은 숫자**를 하나만 남기고 제거하는 문제이다.  
원래 배열의 순서는 유지해야 한다.

---

## 🧠 풀이 아이디어
- 결과를 저장할 리스트를 하나 생성한다.
- 첫 번째 원소는 무조건 결과에 포함시킨다.
- 배열을 순회하며 **이전 값과 현재 값이 다를 경우에만** 결과에 추가한다.
- 결과 리스트를 배열로 변환해 반환한다.

---

## 🔎 핵심 로직
- 이전 값과 현재 값을 비교해 연속 중복 여부 판단
- 중복이 아닐 경우에만 결과 리스트에 추가
- `List<Integer>` → `int[]` 변환

---

## ⏱️ 복잡도 분석
- 시간 복잡도: O(N)
- 공간 복잡도: O(N)

---

## ⚠️ 개선 포인트
- `lastId` 변수 없이 인덱스 비교만으로 구현 가능
