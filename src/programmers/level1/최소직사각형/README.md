# 최소직사각형

## 문제 요약
명함의 가로와 세로 길이가 담긴 2차원 배열 `sizes`가 주어진다.

명함을 지갑에 넣기 위해서는 모든 명함을 수용할 수 있는 직사각형 지갑을 만들어야 한다.  
이때 명함은 회전할 수 있기 때문에 가로와 세로를 서로 바꿔 사용할 수 있다.

모든 명함을 넣을 수 있는 지갑의 최소 크기를 구하고, 그 지갑의 넓이를 반환하는 문제이다.

예시  
`[[60,50],[30,70],[60,30],[80,40]]` → `4000`

---

## 풀이 아이디어
명함은 회전이 가능하기 때문에  각 명함에서 큰 값을 가로 기준, 작은 값을 세로 기준으로 맞추면 된다.

이렇게 정렬된 상태라고 생각하면 지갑의 가로는 모든 명함의 큰 값 중 최대값,  
세로는 모든 명함의 작은 값 중 최대값이 된다.

---

## 핵심 로직
1. 가로 역할을 할 최대값 `colmax` 생성
2. 세로 역할을 할 최대값 `colmin` 생성
3. 명함 배열을 순회하면서
    - `Math.max()`로 두 변 중 큰 값을 구한다
    - `Math.min()`으로 두 변 중 작은 값을 구한다
4. 큰 값들의 최대값을 `colmax`에 저장
5. 작은 값들의 최대값을 `colmin`에 저장
6. 마지막에 `colmax * colmin` 반환

---

## 복잡도

시간 복잡도 : O(N)

---

## 개선 포인트
현재 코드에서는 `Math.max()`와 `Math.min()`을 같은 인덱스에서 두 번 호출하고 있다.

값을 변수에 한 번만 저장해두면 불필요한 계산을 줄이면서 코드도 조금 더 읽기 쉬워질 수 있다.

또 다른 방법으로는 각 명함을 미리 큰 값, 작은 값 순서로 정렬한 뒤  
가로 최대값과 세로 최대값을 구하는 방식으로도 구현할 수 있다.
