# 올바른 괄호

## 🔍 문제 요약
괄호 문자열 `s`가 주어질 때 문자열이 **올바른 괄호 문자열인지 판단하는 문제**이다.  
문자열은 `'('`, `')'` 로만 구성된다.

올바른 괄호 조건
- 열린 괄호와 닫힌 괄호의 짝이 맞아야 한다.
- 닫는 괄호가 먼저 나오면 올바르지 않다.
- 모든 괄호 처리 후 열린 괄호가 남아있으면 안 된다.

---

## 🧠 풀이 아이디어
- 괄호의 짝을 확인하기 위해 **스택(Stack)** 을 사용한다.
- 문자열을 순회하면서
    - `'('` → 스택에 push
    - `')'` → 스택이 비어있지 않으면 pop
- 스택이 비어있는데 `')'` 가 나오면 바로 `false` 반환
- 순회가 끝난 뒤 **스택이 비어있으면 true** 반환

---

## 🔎 핵심 로직
- `'('` → push
- `')'` → pop
- 스택이 비어있는데 `')'` → false
- 마지막에 스택이 비어있는지 확인

---

## ⏱️ 복잡도 분석
- 시간 복잡도: O(N)
- 공간 복잡도: O(N)

---

## ⚠️ 개선 포인트
- 스택 대신 **카운터 변수 하나로도 구현 가능**
- 카운터 방식 사용 시 **공간 복잡도 O(1)** 로 줄일 수 있다.
